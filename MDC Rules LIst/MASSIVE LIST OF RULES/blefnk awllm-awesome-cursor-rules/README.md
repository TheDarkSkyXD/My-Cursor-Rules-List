# blefnk/rules

Neatly composed rules for AI LLMs like Cursor.

## Get Started

1. Install the CLI: `bun -g @reliverse/cli`
2. Run the rules command: `reliverse rules`
3. Select the source: `blefnk/rules`
4. Mark the rules you want to install.
